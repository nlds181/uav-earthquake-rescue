UAV Earthquake Rescue System

This project implements a Learning-Driven Hybrid Adaptive Framework (LD-HAF) for solar-powered UAV swarm coordination in post-earthquake emergency communication.

Features
3D trajectory planning with terrain obstacle avoidance
Adaptive algorithm switching (SGD/Adam/NAG/SGLD/Trust Region/Newton)
Communication coverage heatmap
Solar energy management
Multi-UAV collision avoidance
